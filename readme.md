## SSH Brute Force

- Brute force script with python for SSH servers.

<br>

### How to use

`python2 improve -h` <br>
`python2 -u dev -t 127.0.0.1 -p 2121 -w random.txt`

<br>

### Options

- `u --> Set an specifc User`
- `t --> Set the target`
- `p --> Set the port (Default 22)`
- `W --> Set the wordlist`
- `h --> Help`
